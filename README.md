# linux-script
Kumpulan skrip yang dibuat untuk mempermudah pekerjaan di sistem operasi Linux

## Swapfile_script.sh
Berguna untuk membuat swap RAM dengan cepat  dan efisien.
## Hapus_log.sh
Berguna untuk menghapus log yang menjadi sampah yang ada di dalam direktori /var/log/. Skrip ini akan memilih dengan cerdas file log mana yang boleh dihapus dan mana yang tidak boleh dihapus. Jadi, sistem tidak akan terganggu saat lognya dihapus.
